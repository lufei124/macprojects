# 飞书同步工具

从 MySQL 查询 `generate_lead` 数据，增量写入飞书电子表格。按 **邮箱 + event_time-北京时间** 去重，重复执行不会产生重复数据。

## 执行方式

```bash
# 同步当天数据（默认）
python3 260310推广数据-邮箱明细.py

# 同步指定日期数据
python3 260310推广数据-邮箱明细.py 2026-03-10
```

日期对应 SQL 中 `event_time` 的 **UTC 时间范围**，即 `event_time` 落在该日期 `00:00:00 ~ 23:59:59 UTC` 内的记录。
